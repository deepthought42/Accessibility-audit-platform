# Page Audit Enrichment

Utilities and support scripts for enriching page audit data with additional metadata.

## Overview

This package contains utility resources for the Look-see accessibility audit platform's data enrichment pipeline. It serves as a companion to the `element-enrichment` microservice, providing supplementary tooling for audit data processing.

## Related Services

- **[element-enrichment](../element-enrichment/)** -- The runtime microservice that performs element-level enrichment via headless browser inspection
- **[PageBuilder](../PageBuilder/)** -- Builds the initial page state that enrichment operates on
- **[AuditManager](../AuditManager/)** -- Orchestrates the audit lifecycle that enrichment feeds into
